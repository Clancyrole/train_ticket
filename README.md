# train_ticket
###### 一年一度的天朝大迁徙就要开始了，但是票还是依然的难抢，抢票软件给了钱也抢不到票，人在囧途；为此，博主参加编写了一个12306抢票脚本，抢票再也不用愁

## 运行代码
python3

## 系统运行环境
windows、mac、linux

## 使用工具
chromedriver

## 用户信息输入
在config.ini文件中编写即可，代码无需修改
